# Baseline Monitoring Plan

## Metrics

- request rate
- error rate
- p95 latency
- CPU and memory
- queue depth

## Alerts

- high 5xx rate
- high latency
- worker backlog over threshold
- high CPU

## Dashboards

- service overview
- worker health
- deployment history

## Notes

- Review metrics after each deployment.
- Tune thresholds over time.
